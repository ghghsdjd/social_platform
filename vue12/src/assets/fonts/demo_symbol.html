これは新しい利用方法で、将来の主流であり、プラットフォームでも推奨されているやり方です。詳細はこの記事を参照してください。 この使い方は svg の集合を使っており、他の2つの方法と比べて次の特徴があります：

  • マルチカラーのアイコンをサポートし、単色の制約がなくなります。
  • いくつかのテクニックで、フォントのように font-size や color でスタイルを調整できます。
  • 互換性はやや劣り、IE9+ とモダンブラウザをサポートします。
  • ブラウザの SVG レンダリング性能は一般的に PNG より劣る場合があります。

使用手順は以下のとおりです：

第一歩：プロジェクトで生成された symbol コードを読み込みます：

<script src="./iconfont.js"></script>

第二歩：共通の CSS コードを追加します（一度読み込めば OK）：

<style type="text/css">
.icon {
   width: 1em; height: 1em;
   vertical-align: -0.15em;
   fill: currentColor;
   overflow: hidden;
}
</style>

第三歩：該当するアイコンを選び、クラス名を取得してページに適用します：

<svg class="icon" aria-hidden="true">
  <use xlink:href="#icon-xxx"></use>
</svg>
